# Práctica 2
Este es el archivo README de la práctica 2.

## Contenido
* agente.py: contiene la clase agente que permite guardar un número de objetos para cada agente y generar nombres inventados para ese y para otro agente que actue como oyente.
* juegonombres.py: implementación de la clase agente donde se crean 14 agentes con 7 objetos para cada quien y se realizan 1000 interaciones del juego.

### Ejecutar
Unicamente ejecutar el archivo juegonombres.py para ver el resultado en consola.

### Ejemplo de corrida

python juegonombres.py
